Swap-Tacular

The essence of this application is Craigslist meets Facebook. A cloud bazaar for users to trade, swap, share their things. Minimum viable product is hosted on Heroku.

Features:

  • User Authentification
  • Posts and Comments
  • Google Map Address
  • Uploading Photos

Source code for latest version can be found here. The inspiration for this project came from the sharing culture and more specifically this article.
